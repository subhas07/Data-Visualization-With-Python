# 📊 Data Visualization with Python

## 📌 Objective
Learn and apply data visualization techniques to better understand data patterns and insights.

## 🔍 Topics Covered
- Distribution plots
- Data visualization using Matplotlib
- Understanding data trends and patterns

## 💡 Key Learnings
- Visualizing data helps in better decision making
- Distribution plots reveal hidden patterns
- Graphs are essential before applying Machine Learning

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 🚀 Outcome
Built visual representations of data to support analysis and Machine Learning workflows.
